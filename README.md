# Node.js Tests

Node.js 相关的一些测试及笔记。

在平时学习、工作中，某些知识点或者特性会想要通过一个小的例子进行验证。这个仓库中记录下了这些例子以及相应的笔记，随着时间的推移，我相应内容一定会越来越丰富的。

## 主要内容

仓库中的内容主要分为测试代码以及相应笔记。笔记全部都记录在 [note.md](https://github.com/hezhii/nodejs-tests/blob/master/note.md) 中，小测试则按照文件夹进行了分类。
